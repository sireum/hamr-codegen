HAMR-Codegen is part of [HAMR](https://hamr.sireum.org/), a framework for model-based
development of critical systems. The website is the primary starting point and covers:

* **Install** — installation instructions and prerequisites for the toolchain.
* **Documentation** — the HAMR tour, tutorials, and guides for model-level and
  Rust-based development, plus GUMBO contracts and property-based testing.
* **Overview** — supported modeling languages (AADL, SysMLv2), target languages
  (Rust, C, and Slang), deployment platforms (JVM, Linux, and the seL4 microkernel),
  and the formal-methods integrations (Logika, Verus, and Isabelle).
* **Papers, videos, and slides** — background reading and presentations.
* **Team and Funding** — project sponsors (e.g., DARPA PROVERS/INSPECTA) and contributors.


See the [changelog](changelog.md) for release notes.
